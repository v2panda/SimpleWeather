##How to build
1)  Clone the repository

```
$ git clone https://github.com/v2panda/SimpleWeather.git
```

2) Cd SimpleNote 

```
npm install
```

3) Open in Xcode or 

```
$ react-native run ios
```

3) Compile and run the app in your simulator or iOS device

##Requirements
* Xcode 7.3+
* iOS8+
